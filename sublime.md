- Command + D 选中一个单词
- Command + L 选中一行

- Ctrl+D ->  Ctrl+V 批量替换

- Ctrl+H


- 在GUI下：
>- （1）可按i进入插入模式
>- （2）使用鼠标拖动反选要粘贴的内容，按鼠标左键复制选定块到缓冲区
>- （3）然后将光标移到要粘贴处，按鼠标中键（两键鼠标可同时按左右键），粘贴缓冲区内容。

- 文本终端下：
>- （1）选定文本块，使用v进入可视模式；移动光标键选定内容
>- （2）复制选定块到缓冲区，用y；复制整行，用yy
>- （3）剪切选定块到缓冲区，用d；剪切整行用dd
>- （4）粘贴缓冲区中的内容，用p

- 一编辑窗打开第二个文件，用:sp [filename]
- 个编辑文件之间切换，用Ctrl+w

- 前面加数字表示重复次数，加字母表示使用的缓冲区名称。
- 帮助，用:help [内容或命令]
